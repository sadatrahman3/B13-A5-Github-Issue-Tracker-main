# Answers
1. **What is the difference between `var`, `let`, and `const`?**<br/>
   `var` is function-scoped and hoisted, while `let` and `const` are block-scoped; `const` is for values that won't be reassigned, whereas `let` allows updates.

2. **What is the spread operator (`...`)?**<br/>
   It allows an iterable like an array or object to be expanded into individual elements, which is perfect for making shallow copies or merging data structures.

3. **What is the difference between `map()`, `filter()`, and `forEach()`?**<br/>
   `map` returns a new array of transformed elements, `filter` returns a new array of elements that pass a specific condition, and `forEach` simply executes a function for each item without returning a new array.

4. **What is an arrow function?**<br/>v
   It is a shorter syntax for writing functions using `=>` that does not have its own `this` context, making it very useful for clean, readable callbacks.

5. **What are template literals?**<br/>
   These are strings defined with backticks (`` ` ``) instead of quotes, allowing for easy multi-line strings and "string interpolation" by embedding variables directly using `${}`.
